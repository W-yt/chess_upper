文件说明：
    1、main_project_py是本项目的主文件，用python完成包含项目的运行全部文件
    2、main_project_c是刚开始用c++写的项目文件，包含部分视觉的代码
    3、piece_train_black是本项目中黑色棋子识别的训练用工程
    4、piece_train_red是本项目中红色棋子识别的训练用工程
    5、chess_classify是测试基于cnn的象棋棋子分类效果的工程
    6、image_rotate_c是用c++编写的用于数据集旋转的工程
    7、svm_train是用c++编写的基于opencv内置的svm的棋子分类工程(效果太差)
